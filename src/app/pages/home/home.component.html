Bem-vindo à Aplicação de Gerenciamento de Tarefas

Desenvolva e gerencie suas tarefas pessoais de forma simples e eficiente.

Descrição do Projeto

Este projeto é uma aplicação web simples para o cadastro e gerenciamento de tarefas pessoais. O objetivo é fornecer uma plataforma onde você possa adicionar, visualizar, editar e excluir tarefas com facilidade, além de marcar tarefas como concluídas.

Requisitos Técnicos

  • Frontend (Angular com TypeScript): Formulário de Cadastro de Tarefas, Listagem de Tarefas, Componentização, Roteamento Básico, Design Responsivo.
  • Backend (NodeJS com Express): API RESTful para operações CRUD, Conexão com MySQL, Tratamento de erros simples.
  • Banco de Dados (MySQL): Tabela 'tarefas' com campos como id, título, descrição, data de conclusão e status.

Funcionalidades Extras

  • Filtro de Tarefas por Status (Concluídas ou Pendentes).
  • Ordenação das Tarefas por Data de Conclusão ou Criação.
  • Notificações Simples para Lembrar Tarefas Próximas de Conclusão.
